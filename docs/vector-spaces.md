# Vector spaces and Dirac notation
!!! info "Definition: Linear vector spaces [[1]](#shankar)"
    A **linear vector space** is a collection of objects we call **vectors**, for which there exists two operations:

    1. Vector addition: we can add two vectors together
    2. Scalar multiplication:

Throughout these notes, we will use **Dirac notation**, also known as **Bra-Ket notation**, introduced by Paul Dirac in 1939 [[2]](#dirac-notation).

In Dirac notation, vectors are written using a vertical bar, followed by a label, followed by a right angle bracket: $\ket{\psi}$. In these notes we will usually use Greek letters to label vectors, such as $\psi$ ("psi") or $\phi$ ("phi"), but these choices are arbitrary and you might as well call yours $\ket{\text{Bonnie}}$ or $\ket{\text{Clyde}}$ for all anyone cares.
